# Areilayout

[ZOMEKI](http://arei.jp/)の標準レイアウトを設定します.

## Installation

Add this line to your application's Gemfile:

    gem 'areilayout'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install areilayout

## Usage

#### 1.Change directory

    $ cd RAILS_ROOT
    
#### 2.Type code below

    $ areilayout set -n LAYOUT_NAME -p SOURCE_PATH
