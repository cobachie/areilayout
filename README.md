# Zomekilayout

[ZOMEKI](http://zomeki.jp/)の標準レイアウトを設定します.

## Installation

Add this line to your application's Gemfile:

    gem 'zomekilayout'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install zomekilayout

## Usage

#### 1.cd directory

    $ cd RAILS_ROOT
    
#### 2.Type code below

    $ zomekilayout set SOURCE_PATH LAYOUT_NAME
